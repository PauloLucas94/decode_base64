# 🏛️ BI Auditórios SENAI-SP

Dashboard Power BI com informações dos auditórios das unidades SENAI-SP.

## Estrutura
- `Auditorios_SENAI_SP.xlsx` — base de dados principal
- `converter_simples.py` — converte fotos para Base64
- `fundo_bi_auditorios.png` — plano de fundo do dashboard

## Como usar o conversor
```bash
pip install openpyxl
python converter_simples.py
```

## Tecnologias
- Power BI Desktop
- Python + openpyxl
- SharePoint SENAI-SP
